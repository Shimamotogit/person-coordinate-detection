# Stand & Answer - Camera AI Quiz

カメラで登録した参加者を追跡し、画面上の回答ゾーンへ移動することでクイズに回答できるブラウザアプリです。

## 主な機能

- 複数のカメラから使用するカメラを選択して確定
- 複数人が映っている状態から、丸い番号をクリックして参加者1人を登録
- 登録した参加者だけを追跡・採点
- 人物を矩形で囲まず、参加者の頭上に名前とポイントだけを表示
- 背後の人物へ追跡対象を自動で切り替えない安全な追跡
- 登録者を見失った場合はクイズを停止して再登録
- 時間経過による回答確定
- 管理者のスペースキー押下後、3秒カウントダウン終了時の位置による回答確定
- 2択だけでなく任意個数の選択肢に対応
- JSONファイルから問題を読み込み、問題順をランダム化
- カメラ映像をサーバーへ送信しないブラウザ内処理

## 起動方法

カメラAPIと `fetch()` を利用するため、`file://` で直接開かず、ローカルHTTPサーバーから起動してください。

```bash
python -m http.server 8080
```

ブラウザで `http://localhost:8080` を開きます。

> 本番公開時は HTTPS が必要です。localhost は多くのブラウザで安全なコンテキストとして扱われます。

## 初期設定の流れ

1. 「カメラ一覧を取得」を押し、ブラウザのカメラ利用を許可します。
2. 使用するカメラを選び、「このカメラに決定」を押します。
3. 映像内の人物の頭上に表示される丸い番号から参加者を選択します。
4. 回答確定方法を選び、クイズを開始します。

### 参加者の追跡

参加者は1人だけ登録します。登録時の人物ID、位置、大きさ、縦横比、直前フレームからの移動量を使って追跡します。

登録した人物を一定時間検出できなくなった場合、別の人物を参加者として自動採用しません。クイズを一時停止し、管理者に参加者の再登録を求めます。これにより、参加者の背後に別の人物が立っている場合でも、回答者やポイントが勝手に切り替わることを防ぎます。

## 回答確定方法

### 時間で確定

登録者の足元中央が同じ回答エリアに設定時間以上留まると回答を確定します。設定画面で0.5～10秒の範囲を指定できます。

### スペースキーで確定

管理者がスペースキーを押すと3秒のカウントダウンを開始し、終了時点で登録者が立っている回答エリアを採点します。スペースキーは連打しても二重にカウントダウンしません。

## 問題の追加・編集

`questions.json` を編集します。

```json
{
  "id": "sample",
  "text": "問題文",
  "choices": ["選択肢A", "選択肢B", "選択肢C"],
  "correctIndex": 1
}
```

- `choices`: 2個以上の可変長配列
- `correctIndex`: 正解の選択肢番号（0始まり）

共通設定：

```json
{
  "settings": {
    "pointsPerCorrectAnswer": 100,
    "holdDurationMs": 1200,
    "resultDisplayMs": 1300,
    "spaceCountdownSeconds": 3
  }
}
```

## 判定方法

人物検出には TensorFlow.js / COCO-SSD を利用します。人物検出結果の下辺中央を「足元」とみなし、画面下部を選択肢数で等分した回答エリアに割り当てます。

現在の追跡は、固定カメラで参加者が通常の歩行速度で左右へ移動する用途を想定しています。長時間の完全な遮蔽や参加者同士が重なる状況では再登録が必要になる場合がありますが、別人への自動切替よりも誤採点防止を優先しています。

## 注意事項

- カメラから参加者の全身、特に足元が見えるように配置してください。
- 参加者登録後は、カメラ位置を動かさないでください。
- 暗い場所や人物が小さい場合は検出精度が下がります。
- CDNからTensorFlow.jsとモデルを取得するため、初回起動時はインターネット接続が必要です。
